Control De Empleados

Control de empleados que nos permite editar, crear, eliminar y añadir elementos, permitiendonos añadir imagenes o archivos utilizando la base de datos "sqlite3" y el framework "django" para controlar el backend, utilizado para crear forms e integrarlo en la base de datos, ademas tenemos un diseño responsive y hemos utilizado estilos css para diseñar los gradientes del background y del titulo.

Proyecto Organica

Proyecto Organica es una tienda online con una interfaz limpia, la cual nos permite editar, crear, eliminar y añadir elementos, incorporando esto con una base de datos ralacional "sqlServer" y utilizando python con el framework django para el backend.
